Windows 64 bit with Anaconda Python 2.7, Anaconda Python 3.7, and Intel MKL

Please see run_tests.bat.

Results are sensitive to the CPU and system libraries that may be installed on your machine.

All tests pass on Windows 7 running in vmware on a Macbook Pro 2014.

Requirements:
Python 2.7 (Miniconda installed in %USERPROFILE%/Miniconda2)
Python 2.7 environment installed as python27
Python 3.7 environment installed as python37
Tcl 8.6 installed in the python27 environment
CMAKE installed in the python27 environment
Cygwin 32 bit with diff and perl utilities

The Anaconda prerequisites may be installed using preinstall.bat from the command prompt.
